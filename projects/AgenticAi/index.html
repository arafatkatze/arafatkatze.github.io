Agentic search

Agentic search is a very difficult unsolved problem and has historically had all different kinds of strategies. The most common ones are Grep and Vector search but each have their pitfalls. Its still unclear to me what the actual answers are here

Cursor vs. Cognition have opposite takes on agent search:

- Cursor: Custom embeddings trained on agent traces improve accuracy by 12.5%
- Cognition: Embeddings are counterproductive, so we trained models to use grep with 8x parallel tool calls.

Both have benchmarks.
