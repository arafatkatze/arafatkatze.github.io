## Running it locally

This is a [Jekyll](https://jekyllrb.com/) site. To preview it on your machine:

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000>.

Prefer Docker? There's a `docker-compose.yml` for a zero-setup local build:

```bash
docker compose up
```

Content lives in `_pages/`, `_posts/`, and `_projects/`; layout and styling live in `_layouts/`, `_includes/`, and `_sass/`; site-wide settings are in `_config.yml`.
